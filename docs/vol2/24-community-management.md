### 24.2 — Moderation Guidelines  
  
**Rules for @bakstab_news and in-game chat:**  
1. No spam.  
2. No personal attacks (on real people — attacking characters is fine and encouraged).  
3. No political extremism (satire of fuel prices = OK; calls for violence = ban).  
4. No cheating accusations without evidence (take it to DMs).  
5. English is OK, but Russian is primary.  
  
**Ban Tiers:**  
- Warning (1st offense).  
- 1-hour mute (2nd).  
- 24-hour mute (3rd).  
- Permanent ban (4th, or severe offense).  
  
### 24.3 — Bug Bounty Program  
  
**Tiers:**  
- **Minor bug** (typo, visual glitch): 50 Бабки.  
- **Moderate bug** (task doesn't complete, UI overlap): 200 Бабки.  
- **Major bug** (match crash, disconnect): 500 Бабки + "Тестировщик" title.  
- **Critical bug** (server crash, exploit, data loss): 2000 Бабки + "Герой Двора" title.  
  
**Submission:** Player sends a screenshot/video + description to @bakstab_support. Manual review within 48 hours.  
  
### 24.4 — Content Creator Program  
  
**Tier 1 — Micro Creators (1k-10k followers):**  
- Exclusive "Инфлюенсер" hat.  
- Early access to seasonal content (1 week before launch).  
- Shoutout in @bakstab_news.  
  
**Tier 2 — Mid Creators (10k-100k):**  
- All Tier 1 benefits.  
- Custom in-game cosmetic (their likeness as a character).  
- Revenue share on referred players' first purchase (10%, via referral tracking).  
  
**Tier 3 — Macro Creators (100k+):**  
- All Tier 2 benefits.  
- Branded cosmetic line (their brand as hats/pets).  
- Co-streamed events.  
  
**Outreach:**  
- Find creators via TikTok/Telegram search for "игры telegram", "among us", "мобильные игры".  
- DM them: "Hey, I made a game I think you'd like. Free exclusive skin if you play it on stream. [link]"  
  
### 24.5 — Player Feedback Loop  
  
**Feedback Channels:**  
1. In-game "Пожаловаться/Предложить" button (sends to @bakstab_support).  
2. @bakstab_news comments.  
3. GitHub Issues (public, for bug reports).  
  
**Processing:**  
- All feedback logged in a spreadsheet (date, player, category, text).  
- Weekly review by the team (us).  
- Top 3 requested features each month are considered for the next patch.  
- Players whose suggestions are implemented get 500 Бабки + "Вдохновитель" title.
